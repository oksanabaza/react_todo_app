## Installation

Use the package:

cd react_todo_app
```bash
npm i express
```
cd react_todo_app
```bash
npm init
```
cd client
```bash
npm install
```
-------------------------
start concurrently 
```bash
npm run dev
```
start client
```bash
npm run start
```
start server
```bash
npm run server
```
